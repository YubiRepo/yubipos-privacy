Kebijakan Privasi Yubi POS

Terakhir Diperbarui: 11 Maret 2026

Terima kasih telah memilih Yubi POS sebagai mitra manajemen bisnis Anda. Kami berkomitmen penuh untuk melindungi data pribadi dan transaksi bisnis Anda.

1. Informasi yang Kami Kumpulkan

Untuk memberikan layanan Point of Sale yang optimal, kami mengumpulkan beberapa data berikut:

  • Informasi Pengguna: Nama lengkap, alamat email, dan nomor WhatsApp untuk verifikasi akun.
  • Data Transaksi: Detail penjualan, stok barang, harga produk, dan kategori usaha.
  • Data Pelanggan: Nomor telepon pelanggan yang Anda masukkan untuk keperluan pengiriman struk digital.
  • Log Perangkat: Alamat IP, versi OS, dan model perangkat untuk meningkatkan stabilitas aplikasi.

2. Penggunaan Data Anda

Kami hanya menggunakan data Anda untuk kepentingan fungsionalitas aplikasi:

  • Memproses registrasi dan login menggunakan sistem OTP WhatsApp.
  • Mengelola operasional harian toko dan laporan keuangan.
  • Mengirimkan Digital Receipt langsung ke WhatsApp pelanggan.
  • Menganalisis performa aplikasi guna memperbaiki kendala teknis.

3. Layanan Pihak Ketiga

Yubi POS bekerja sama dengan penyedia layanan pihak ketiga tertentu:

  • Fonnte: Gateway pesan WhatsApp. Data nomor tujuan diteruskan hanya saat pengiriman pesan dilakukan.
  • Infrastruktur: Data disimpan secara aman pada layanan VPS yang berlokasi di Indonesia.

4. Keamanan Informasi

Kami mengimplementasikan protokol keamanan SSL (Secure Sockets Layer) untuk setiap komunikasi data. Data sensitif seperti password disimpan menggunakan enkripsi satu arah yang aman.

5. Akses dan Izin Perangkat

Aplikasi Yubi POS memerlukan beberapa izin fungsional:

  • Internet: Untuk sinkronisasi data ke cloud server.
  • Bluetooth: Untuk koneksi dengan printer thermal struk.
  • Penyimpanan: Untuk menyimpan sementara laporan atau struk format PDF.

6. Penghapusan Data

Anda memiliki hak penuh atas data Anda. Jika Anda ingin berhenti menggunakan layanan kami, Anda dapat mengajukan permohonan penghapusan akun melalui layanan dukungan kami.
