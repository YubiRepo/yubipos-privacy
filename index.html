Kebijakan Privasi Yubi POS

Terakhir Diperbarui: 11 Maret 2026

Terima kasih telah memilih Yubi POS sebagai mitra manajemen bisnis Anda. Kami sangat menghargai kepercayaan Anda dan berkomitmen penuh untuk melindungi data pribadi pengguna serta data transaksi bisnis Anda.

1. Informasi yang Kami Kumpulkan

Untuk memberikan layanan Point of Sale yang optimal, kami mengumpulkan beberapa data berikut:

  • Informasi Pengguna: Nama lengkap, alamat email, dan nomor WhatsApp yang digunakan untuk verifikasi akun dan identitas merchant.
  • Data Transaksi: Informasi detail mengenai penjualan, stok barang, harga produk, dan kategori usaha yang Anda kelola melalui aplikasi.
  • Data Pelanggan: Informasi nomor telepon pelanggan yang Anda masukkan secara sukarela untuk keperluan pengiriman struk digital.
  • Log Perangkat: Informasi teknis seperti alamat IP, versi sistem operasi Android, dan model perangkat untuk meningkatkan stabilitas aplikasi Flutter kami.

2. Penggunaan Data Anda

Kami hanya menggunakan data Anda untuk kepentingan fungsionalitas aplikasi, antara lain:

  • Memproses registrasi dan login menggunakan sistem OTP WhatsApp.
  • Mengelola operasional harian toko, termasuk pencatatan transaksi dan laporan keuangan.
  • Mengirimkan Digital Receipt (Struk Online) langsung ke WhatsApp pelanggan Anda.
  • Menganalisis performa aplikasi guna memperbaiki bug atau kendala teknis lainnya.

2. Layanan Pihak Ketiga

Yubi POS bekerja sama dengan penyedia layanan pihak ketiga untuk menjalankan fungsi spesifik:

  • Fonnte: Kami menggunakan API Fonnte sebagai pengirim (gateway) pesan WhatsApp. Data nomor tujuan akan diteruskan ke server Fonnte secara terenkripsi hanya saat pengiriman pesan dilakukan.
  • Infrastructure: Data Anda disimpan secara aman pada layanan VPS (Virtual Private Server) yang berlokasi di pusat data Indonesia.

4. Keamanan Informasi

Kami mengimplementasikan protokol keamanan SSL (Secure Sockets Layer) untuk setiap komunikasi data antara aplikasi mobile dan server backend Laravel kami. Data sensitif seperti password disimpan menggunakan enkripsi satu arah yang tidak dapat dibaca kembali.

5. Akses dan Izin Perangkat

Aplikasi Yubi POS mungkin memerlukan beberapa izin pada perangkat Android Anda, seperti:

  • Internet: Untuk sinkronisasi data ke cloud server.
  • Bluetooth: Untuk menghubungkan aplikasi dengan printer thermal struk (jika digunakan).
  • Storage: Untuk menyimpan sementara laporan penjualan atau struk dalam format PDF sebelum dibagikan.

6. Penghapusan Data

Anda memiliki hak penuh atas data Anda. Jika Anda memutuskan untuk berhenti menggunakan layanan kami, Anda dapat mengajukan permohonan penghapusan data akun dan seluruh data bisnis Anda dengan menghubungi layanan dukungan kami.
